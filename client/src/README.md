# Notes App with Tagging

## Features
- Create notes
- Add tags
- Search notes by tags
- Edit notes
- Delete notes

## Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- MongoDB Atlas (Database)

## How to run

### Backend
cd server
npm install
node server.js

### Frontend
cd client
npm install
npm run dev
